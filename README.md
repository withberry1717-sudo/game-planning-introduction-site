## 更新方針

このサイトは、ゲームプランナー志望としての制作物・分析・企画書をまとめる入口です。

- `Monster Destroyer`：制作振り返り兼紹介サイトへの導線
- `アクションゲーム分析Web`：人気タイトルのアクションをフレーム単位で分析したページへの導線
- `Unreal Engine向けアクションゲーム企画書`：完成後にPDFまたはWebページへのリンクを掲載予定

リンクを1つしか提出できない企業にも、このサイトを入口として渡せるように構成しています。
